# Website Migration Checklist

Essential steps to ensure your website migration doesn't harm your SEO rankings and maintains search visibility.

## Pre-Migration Planning
- [ ] Audit current website performance
- [ ] Document current URL structure
- [ ] Create comprehensive redirect mapping
- [ ] Set up staging environment

## During Migration
- [ ] Implement 301 redirects properly
- [ ] Update internal links
- [ ] Test all forms and functionality
- [ ] Verify mobile responsiveness

## Post-Migration Tasks
- [ ] Monitor for 404 errors
- [ ] Update XML sitemaps
- [ ] Resubmit to search engines
- [ ] Monitor rankings and traffic

## Technical Considerations
- [ ] Preserve URL structure when possible
- [ ] Maintain page load speeds
- [ ] Keep meta tags and structured data
- [ ] Test Core Web Vitals

## Monitoring & Recovery
- [ ] Set up monitoring alerts
- [ ] Track key performance indicators
- [ ] Prepare rollback plan
- [ ] Document lessons learned
